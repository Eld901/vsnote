###### 安装 you-get  
1. you-get中文说明：https://github.com/soimort/you-get/wiki  
2. 环境配置 安装ffmpeg：http://ffmpeg.org/  
复制bin目录下文件到本地文件夹 复制本地文件夹路径配置环境变量  
新建变量名ffmpeg 变量值为文件夹路径 打开cmd输入ffmpeg验证安装成功  
3. 环境配置 安装python：https://www.python.org/downloads/  
提前勾选AddToPath 打开cmd输入python验证安装成功  
4. 命令行安装youget：  
安装命令`pip3 install you-get` 按提示输入命令`python -m pip install --update pip`  
准备下载视频输入命令`you-get` 下载视频输入命令`you-get -o 本地目标路径 视频网页地址`  
更新命令`pip3 install --upgrade you-get`
